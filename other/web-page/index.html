Bridge measurements database

Welcome to the Structural Dynamics measurement database. Here, you can download recordings of monitoring projects conducted by the Structural Dynamics Group at the Norwegian University of Science and Technology (NTNU).

Currently, there are data from the monitoring projects on the Hardanger Bridge (suspension bridge) and the Bergs√∏ysund Bridge (floating bridge) available. Both raw data, processed (and resampled) data and statistical information from the recordings are available for download.

Format

The files are stored in mat-format, but supporting code snippets are available to enable import in a Python environment.
